自然语言处理第一次作业

对 20000 字左右的医学语料进行分词，词性标注和医学命名实体识别的加工。

使用 pkuseg 和 thulac 两款分词库进行分词，然后编写程序生成两者结果的对比结果（ 如图3），生成 html 文件，通过 tag 标记和 css 高亮分词与词性标注的不同处。

然后根据 北京大学现代汉语语料库基本加工规范（2002版） 与 医学语料命名实体识别加工规范 进行人工修正，和人工实体识别标注。
